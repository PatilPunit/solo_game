**🏹 Demon Hunter: Dungeon RPG (C Language)**

“Step into the shadows… fight demons… rise as the strongest hunter.”

**🌍 Overview**

Demon Hunter is a console-based RPG game built in C language where the player embarks on a journey to become a powerful demon hunter.
You explore dungeons, fight enemies, earn rewards, manage inventory, gamble in casino games, attend classes, and even travel across countries 🌎

This project blends:

**🎮 Game logic**

📊 Player progression system
🧠 Quiz & learning mechanics
💰 Economy (shop, bank, selling)
⚔️ Combat system
✨ Features

**⚔️ Combat System**

Fight enemies across ranks:
D, C, B, A, S rank
Dynamic HP-based battles
Weapons & armor affect damage
Potions for healing and attack
Dungeon Exploration
Choose difficulty level
Fight random enemies

**Earn:**
🔮 Crystals
💠 Mega Gems
✨ Experience

**Track:**

Buy items like:

System	Description
